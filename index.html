Programação Front-End

O que é Programação Front-End?

A programação front-end é a parte do desenvolvimento web responsável pela construção da interface do usuário. É a camada que o usuário final interage, incluindo o layout, design e interatividade.

Curiosidades

  1. O primeiro site da história foi criado em 1991 por Tim Berners-Lee, o inventor da World Wide Web.
  2. O primeiro navegador web foi o WorldWideWeb, criado em 1990 também por Tim Berners-Lee.
  3. O HTML foi criado em 1989 por Tim Berners-Lee para facilitar a criação de documentos online.

Tags HTML Comuns

Algumas das tags HTML comuns utilizadas na programação front-end:

  • <h1> a <h6>: Cria cabeçalhos com diferentes tamanhos.
  • <p>: Cria parágrafos de texto.
  • <mark>: Destaca um texto.
  • <small>: Define um texto pequeno.
  • <i>: Cria um texto em itálico.
  • <u>: Sublinha um texto.
  • <strong>: Define um texto em negrito.
  • <ol>: Cria uma lista numerada.
  • <ul>: Cria uma lista não numerada.
  • <li>: Define um item de lista.
  • <a>: Cria um link.
  • <hr>: Cria uma linha horizontal.
  • <sub>: Define um texto em subscrito.
  • <sup>: Define um texto em sobrescrito.
  • <blockquote>: Define um bloco de citação.
  • <font>: Define o estilo da fonte do texto.
  • <del>: Define um texto que foi removido.
  • <abbr>: Define uma abreviação.

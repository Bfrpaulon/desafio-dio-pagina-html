Curso Básico de HTML5

O Curso Básico de HTML5 tem como objetivo ensinar aos participantes as principais tags do HTML5 e como utilizá-las para desenvolver páginas web simples.

  1. Introdução ao HTML5
  2. Tags básicas do HTML5, como <html>, <head>, <body>, <h1>, <p>, entre outras.
  3. Criação de links com a tag <a>
  4. Listas com as tags <ol> e <ul>
  5. Uso de imagens com a tag <img>

Curso Avançado de HTML5

O Curso Avançado de HTML5 é indicado para aqueles que já possuem conhecimentos básicos em HTML5 e desejam aprofundar seus estudos.

  • Uso de formulários com as tags <form>, <input>, <textarea>, entre outras.
  • Estilos CSS em HTML5 com as tags <style> e <link>
  • Criação de tabelas com a tag <table>
  • Uso de áudio e vídeo com as tags <audio> e <video>

Curso Master de HTML5

O Curso Master de HTML5 é o mais completo e abrange desde os conceitos básicos até os mais avançados de HTML5.

O HTML5 é uma linguagem de marcação que permite a criação de páginas web mais modernas e interativas. Com o Curso Master de HTML5, você irá dominar essa tecnologia e estar pronto para criar sites incríveis.

Equipe de Instrutores do Curso Master de HTML5
  • Conceitos avançados de HTML5
  • Uso de canvas com a tag
  • Uso de geolocalização com a tag
  • Criação de animações com a tag
  • Uso de microdados com a tag
  • Uso de web workers com a tag
  • Uso de websockets com a tag
  • Introdução ao desenvolvimento responsivo com a tag

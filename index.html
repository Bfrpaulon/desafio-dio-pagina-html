Sobre o Curso

O Curso de Formação HTML5 tem como objetivo capacitar os participantes a desenvolverem páginas web utilizando a linguagem HTML5.

O curso é indicado para iniciantes na área de desenvolvimento web e não requer conhecimentos prévios em programação.

Objetivos do Curso

  • Aprender a estrutura básica do HTML5
  • Conhecer as principais tags do HTML5
  • Aprender a utilizar formulários em HTML5
  • Desenvolver páginas web responsivas
